THEORY
Data transformation is the process of converting raw data into a structured and meaningful format for better analysis. One important transformation technique is data binning, where continuous numerical values are grouped into meaningful categories such as low, medium, and high. Pandas provides powerful functions for binning, formatting text, changing data types, rounding numerical values, sorting records, and analyzing unique categories. These techniques are widely used in business analytics, sales analysis, and operational data processing to simplify interpretation and improve decision-making.

One-line explanation of functions used:

A -> DataFrame() – Creates a structured tabular dataset from dictionary data.

B -> cut() – Converts continuous numerical values into categorical bins.

C -> astype() – Changes the data type of a selected column.

D -> dtypes – Displays the data types of all columns.

E -> str.upper() – Converts text values in a column to uppercase.

F -> str.lower() – Converts text values in a column to lowercase.

G -> round() – Rounds numerical values to the specified number of decimal places.

H -> sort_values() – Sorts the dataset based on selected columns.

I -> unique() – Returns all unique values present in a column.

J -> value_counts() – Counts the frequency of each category.

K -> print() – Displays the transformed dataset or output values.
